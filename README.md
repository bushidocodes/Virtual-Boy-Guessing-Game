# Guessing Game

A simple guessing game web application using a vector art style inspired by the Nintendo Virtual Boy.

## Setup

```sh
npm install
```

## Running

```sh
npm start
```

Opens the game at http://localhost:3000.

## Testing

```sh
npm test
```

Runs the Vitest unit suite headlessly in Node.

```sh
npm run typecheck
```

Type-checks the TypeScript sources with `tsc --noEmit`.

## Building

```sh
npm run build
```

Builds the production bundle to `dist/` with Vite.
